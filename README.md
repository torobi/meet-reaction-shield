# Meet Reaction Shield

A Chrome extension to control Google Meet reactions as you like.

---

## Features

- **Hide** specific reaction in Google Meet
- Show a **confirmation dialog** before sending selected reactions
- Intuitive UI: easily configure with clicks or keyboard navigation
